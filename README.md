# InsureIQ – Insurance Cost Predictor

## 📌 Project Overview
This project predicts insurance cost using Machine Learning.

## 🚀 Features
- Upload dataset
- Train model
- Predict insurance cost
- User-friendly Streamlit interface

## 🛠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit

## ▶️ How to Run
1. Install requirements
2. Run: streamlit run app.py
